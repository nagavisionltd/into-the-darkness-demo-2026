🌑 LEVEL 1: NOKTARA - MEMORY MAZE
Theme: Psychological horror in a burning city that shifts between reality and nightmare
🎯 CORE MECHANICS
Memory Shards System:
Truth Shards (Blue): Reveal safe paths, temporary platforms, slow time
False Shards (Red): Create illusions, spawn traps, distort controls
Reality Shift: Level geometry changes based on shard collection
Memory Echoes: Ghost replays of past events show hints or mislead
📐 ROOM-BY-ROOM BREAKDOWN (14 Rooms Total)
SECTION 1: INTRODUCTION (Rooms 1-3)
Room 1: "The Awakening"
Type: Horizontal scroll (safe intro)
Layout: Burning street, scattered debris
Teaching:
Basic movement through destroyed city
First Truth Shard (blue) floating visibly → reveals ghostly platform
Memory Echo: Shows character's past self running safely (tutorial hint)
Enemies: None (pure exploration)
Visual: Fires, full moon, distant demon silhouettes
Duration: 45 seconds
Room 2: "First Contact"
Type: Horizontal → slight vertical climb
Layout: Collapsed building creates stairs
New Element: False Shard (red) → spawns Shadow Stalker enemy
Combat Intro:
1x Shadow Stalker (teleports behind player)
Teaches: Fast attacks counter teleport (from your design doc)
Knockback Practice: Launch enemy into fire pit (+20 EC bonus)
Duration: 90 seconds
Room 3: "The Choice"
Type: One-screen box (decision point)
Layout: Two paths visible:
Left: Truth Shard visible, safe route
Right: False Shard visible, harder route
Mechanic: Player chooses which shard to collect first
Truth → unlocks blue path forward
False → spawns 2x Shadow Stalkers, but drops Ethics Credits
Design: No wrong choice, just different challenge
Duration: 60-120 seconds (skill-based)
SECTION 2: ESCALATION (Rooms 4-7)
Room 4: "Shifting Streets" ⭐ MAIN GIMMICK DEBUT
Type: Horizontal scroll with reality shifts
Layout: Street that changes when you collect shards
Core Mechanic:
Collect Truth Shard → buildings reform, create platforms
Collect False Shard → buildings crumble, pits open
Challenge: Must collect Truth to proceed, False creates shortcuts with enemies
Visual: Screen shake (0.004 intensity) when reality shifts
Duration: 2 minutes
Room 5: "Echo Chamber"
Type: Vertical climb
Layout: Multi-story burning building interior
New Element: Memory Echoes as Platforms
Ghost replays appear as semi-solid platforms
Must time jumps to when echoes "materialize"
Enemies: 2x Memory Leech (drain XP, weak to area attacks)
Combat Twist: Knock them into fires before they drain you
Particle Effects: Blue particles (Truth) vs red particles (False)
Duration: 2 minutes
Room 6: "The Gauntlet"
Type: Horizontal scroll (combat focus)
Layout: Wide street, multiple enemy spawn points
Challenge:
Wave 1: 3x Shadow Stalkers
Wave 2: 2x Memory Leech + 1x Illusion Knight (fake copies)
Wave 3: Mixed group (all 3 types)
Reward: Breakable objects (10-30 EC each) after clearing
Hit Stop: 100ms freeze on finisher hits (combo hit 7+)
Duration: 3 minutes
Room 7: "Breather - Memory Fragment" 🧘
Type: One-screen box (safe zone)
Layout: Abandoned apartment, no enemies
Purpose:
Story moment: Character finds photo/memory
Checkpoint (if you implement saves)
Teaches: Lore about Noktara's fall
Visual: Calm, no fires, just moonlight through window
Duration: 30 seconds (player-controlled pacing)
SECTION 3: MASTERY (Rooms 8-11)
Room 8: "Truth & Lies" (Echo of Room 3)
Type: Hybrid (zig-zag descent)
Layout: Building interior collapsing downward
Evolution: Now BOTH shard types appear simultaneously
Challenge:
Must collect Truth to open door
False shards tempt with EC rewards but spawn enemies
Time pressure: Floor breaks every 10 seconds
Skill Test: Can you grab False shard AND Truth shard safely?
Duration: 90 seconds
Room 9: "Shadow Dance"
Type: Vertical descent (danger zone)
Layout: Elevator shaft, breaking platforms
Enemies: 4x Shadow Stalkers (teleport constantly)
Mechanic:
Platforms disappear after 2 seconds
Must use Truth Shard power (slow time) to navigate
False Shards create fake platforms (fall through)
Screen Shake: 0.006 intensity on platform breaks
Duration: 2 minutes
Room 10: "The Mirror" 🔥 SECRET TEST
Type: One-screen box (optional challenge)
Visual Tell: Glowing blue door (easy to miss)
Requirement: Must have collected 3+ Truth Shards
Challenge:
Fight your "dark reflection" (enemy with your moves)
Uses same combat system (knockback scaling, hit stop)
Reward:
Memory Shard (permanent +10 HP or +5% damage)
100 EC
Design: Skippable if player misses the door
Duration: 2-4 minutes (skill-based)
Room 11: "Convergence"
Type: Horizontal → vertical hybrid
Layout: Street leading to massive cathedral
All Mechanics Combined:
Reality shifts mid-combat
Memory Echoes as platforms AND enemies
Truth/False shard choices affect enemy spawns
2x Illusion Knight + 3x Memory Leech + 2x Shadow Stalker
Pre-Boss Gauntlet: Tests everything learned
Particle Bursts: 12-20 particles on heavy hits (ADD blend mode)
Duration: 3-4 minutes
SECTION 4: CLIMAX (Rooms 12-14)
Room 12: "Antechamber"
Type: Horizontal (breather before boss)
Layout: Cathedral entrance, dramatic lighting
Purpose:
Final Truth/False choice (affects boss difficulty)
Truth → boss has less HP
False → boss hits harder but drops more EC
Visual: Massive doors, ominous music swell
Duration: 30 seconds
Room 13: "BOSS: Memory Keeper" 👹
Type: Arena (one-screen box)
Boss Design:
Phase 1: Uses Truth powers (creates platforms, slows time)
Phase 2 (50% HP): Uses False powers (illusions, traps)
Phase 3 (25% HP): Both powers simultaneously
Mechanics:
Knockback scaling: 2.5x on finishers (launches boss)
Hit stop: 100ms on heavy attacks
Screen shake: 0.008 intensity on boss slams
Weakness:
Player must use opposite shard type to counter boss
Truth boss → use False shard attacks
False boss → use Truth shard attacks
Reward:
Noktara Memory Core (unlocks V Blast ability)
200 EC
Access to Level 2 (Khemra)
Duration: 5-7 minutes
Room 14: "Aftermath"
Type: Horizontal (exit)
Layout: Cathedral roof, dawn breaking
Purpose:
Victory moment
Transition to Level 2
Shows city still burning (ongoing quest)
Visual: Sun rising, character silhouette
Duration: 20 seconds
